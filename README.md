# 🌍 Universal Translator API

A modern, full-stack translation tool with auto-language detection and multi-server fallback. Built using **Node.js, Express, and Axios**.

---

## ✨ Features
* **35+ Languages Support**: Translate text between dozens of global languages.
* **Auto-Detection**: Automatically identifies the source language.
* **Multi-Server Fallback**: If one translation mirror is down, it automatically tries another.
* **MVC Architecture**: Clean folder structure for easy maintenance.

---

## 📁 Project Structure
* **controllers/**: Handles translation requests and responses.
* **services/translateService.js**: Core logic that connects to translation mirrors.
* **routes/translateRoutes.js**: API endpoint definitions.
* **server.js**: Main entry point for the Express server.
